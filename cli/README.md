# TUI Module - 终端界面模块

## 概述

TUI 模块是用户层的核心组件，提供丰富的交互式终端界面，处理用户输入输出，支持设置向导和多种命令模式。

## 🏗️ 架构设计

### 模块结构

```
cli/
├── __init__.py
├── main.py           # TUI 主入口
├── ui.py             # UI 组件和主题
├── selector.py       # 交互式选择器
└── setup_wizard.py   # 设置向导
```

## 🧩 核心组件

### 1. TUI 主入口 (cli.py / cli/main.py)

**职责**: 终端参数解析和命令分发。

**支持的命令**:

| 命令 | 功能 | 示例 |
|------|------|------|
| `chat` | 启动交互式对话 | `python cli.py chat` |
| `setup` | 运行设置向导 | `python cli.py setup` |
| `sessions` | 列出所有会话 | `python cli.py sessions` |
| `batch` | 批量处理 | `python cli.py batch -i input.json` |
| `version` | 显示版本信息 | `python cli.py version` |

### 2. UI 组件 (cli/ui.py)

**职责**: 终端界面渲染，包含主题系统和输出格式化。

**主题系统**:

```python
class Theme:
    # 极客绿主题
    PRIMARY = "\033[38;2;34;197;94m"      # Emerald green
    PRIMARY_BRIGHT = "\033[38;2;48;211;99m"
    PRIMARY_DIM = "\033[38;2;22;130;62m"
    
    # 其他颜色
    ACCENT = "\033[38;2;6;182;212m"       # Cyan
    WARNING = "\033[38;2;245;158;11m"     # Amber
    ERROR = "\033[38;2;239;68;68m"        # Red
    SUCCESS = "\033[38;2;34;197;94m"      # Green
```

**UI 组件**:
- `print_header()` - 打印头部信息
- `print_prompt()` - 打印输入提示
- `print_response()` - 打印响应
- `print_menu()` - 打印菜单
- `print_status()` - 打印状态栏

### 3. 设置向导 (cli/setup_wizard.py)

**职责**: 首次运行时引导用户配置大模型信息。

**设置流程**:

```
开始 → 选择提供商 → 配置 API Key → 选择模型 → 保存配置 → 完成
         │              │              │
         ▼              ▼              ▼
    25+ 提供商        密钥输入       模型列表
```

**配置项**:
- LLM 提供商选择
- API Key 输入
- 模型选择
- API 地址配置（可选）

### 4. 交互式选择器 (cli/selector.py)

**职责**: 提供列表选择功能，支持键盘操作。

**特性**:
- 数字输入选择（Windows 兼容）
- 支持单选和多选
- 高亮显示选项

## 🎯 使用示例

```bash
# 启动交互式对话
python cli.py chat

# 运行设置向导
python cli.py setup

# 查看会话列表
python cli.py sessions

# 批量处理
python cli.py batch -i tasks.json -o results.json
```
