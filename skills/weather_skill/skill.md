# Weather Skill

## 概述
查询天气信息的技能，支持获取实时天气和天气预报。

## 技能信息
- **ID**: weather
- **名称**: Weather
- **分类**: tools
- **描述**: 查询指定城市的天气信息

## 参数
| 参数名 | 类型 | 必填 | 描述 | 示例 |
|--------|------|------|------|------|
| city | string | 是 | 城市名称 | 北京 |
| days | int | 否 | 预报天数（默认1天） | 3 |

## 别名
- weather
- 查询天气
- 获取天气

## 示例
- "查询北京天气"
- "北京今天天气怎么样"
- "上海未来3天天气预报"

## 权限
- 不需要特殊权限

## 依赖
- 需要网络连接
- 依赖外部天气 API
