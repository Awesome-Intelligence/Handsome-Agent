# Tools Module - 工具系统

## 📋 概述

工具系统让 Agent 能够帮你完成各种任务，包括读写文件、执行命令、搜索信息等。

## 🧩 支持的工具

| 类别 | 功能 |
|------|------|
| **📁 文件** | 读写文件、创建目录、管理文件 |
| **🖥️ 命令** | 执行终端命令、运行代码、使用 Git |
| **🌐 网络** | 搜索网页、抓取内容、发送请求 |
| **💻 代码** | 代码分析和格式化 |
| **🌐 浏览器** | 浏览器自动化操作 |
| **🎨 多媒体** | 生成图片、文字转语音 |
| **✅ 任务** | 管理 Todo 列表 |
| **💾 记忆** | 存储和检索长期记忆 |
| **📋 技能** | 管理和执行技能 |
| **👁️ 视觉** | 图片分析和 OCR 识别 |
| **📅 定时** | 定时任务调度 |
| **🚀 应用** | 启动应用程序 |

## 🏛️ 工作原理

```
你说"帮我搜索最新的 AI 新闻"
        ↓
   Agent 理解你的意图
        ↓
   自动选择合适工具
        ↓
   执行搜索并返回结果
```

## 🔒 安全机制

- 命令执行有安全限制
- 文件操作有权限控制
- 超时防止程序卡死
- 敏感信息自动脱敏
