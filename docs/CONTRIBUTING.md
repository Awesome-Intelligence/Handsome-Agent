# Contributing to Handsome Agent

## Development Setup

1. Fork the repository
2. Clone your fork
3. Create a virtual environment
4. Run tests: `python verify.py`

## Code Style

- Follow PEP 8
- Use type hints
- Add docstrings

## Pull Request Process

1. Create a feature branch
2. Write tests
3. Submit PR with description

## Reporting Issues

- Use GitHub Issues
- Include minimal reproduction
- Specify environment
