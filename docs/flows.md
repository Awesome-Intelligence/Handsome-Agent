## Flow 1: 简单对话

**触发场景**: 用户发起闲聊、问天气、问时间等不需要工具的问题

### 用户示例

```
用户: "你好，今天天气怎么样？"
用户: "现在几点了？"
用户: "给我讲个笑话吧"
```

### 完整流程

```
┌──────────────────────────────────────────────────────────────────────────────┐
│ 🚪 Access Layer / 💬 CLI                                                     │
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ cli/main.py                                                             │ │
│ │ Step 1: 接收用户输入，验证非空，传递给 Agent                      │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 🚪 Access Layer / 🚪 Gateway                                                 │
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ agent/agent.py                                                   │ │
│ │ Step 2: Session 模块记录用户输入                                        │ │
│ │         - 将消息添加到 messages 列表                                   │ │
│ │         - 记录 timestamp                                                │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 🧠 Decision Layer / 🤖 LLM                                                  │
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ agent/tool_selector/llm_tool_selector.py                                           │ │
│ │ Step 3: LLM 分析意图，判断是否需要工具                                   │ │
│ │         输入: 用户输入 + 可用工具列表 + 历史对话                         │ │
│ │         输出: Decision {type: "direct_response", tool: null}            │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 🧠 Decision Layer / 🤖 LLM                                                  │
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ agent/llm/openai_provider.py (或其他 LLM Provider)                      │ │
│ │ Step 4: LLM 生成回复                                                   │ │
│ │         输入: 完整对话上下文                                            │ │
│ │         输出: LLMResponse {content, model, latency_ms, usage}           │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 🚪 Access Layer / 🚪 Gateway                                                 │
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ agent/modern_agent.py                                                   │ │
│ │ Step 5: Session 记录回复，返回给 CLI，展示给用户                         │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
```

### 日志输出示例

```

2026-06-01 10:30:02 - 🚪 [Access] - INFO - (Agent) Response generated in 2.00s
```

**触发场景**: 用户要求查看文件内容

### 用户示例

```
用户: "帮我看看 agent.md 的内容"
用户: "打开 config.json 看看配置"
用户: "显示 README.md 的前 100 行"
```

### 完整流程

```
┌──────────────────────────────────────────────────────────────────────────────┐
│ 🧠 Decision Layer / 🤖 LLM                                                  │
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ agent/llm_tool_selector.py                                              │ │
│ │ Step 1: LLM 决策                                                        │ │
│ │         Decision {type: "tool_call", tool: "read_file"}                │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 🏃 Execution Layer / 🛠️ ToolExec                                            │
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ tools/registry.py                                                       │ │
│ │ Step 2: 工具参数验证                                                    │ │
│ │         - 验证工具存在: read_file                                        │ │
│ │         - 验证参数完整: file_path                                        │ │
│ │         - 安全检查: 禁止读取 ~/.agent_z 等敏感目录                │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 🏃 Execution Layer / 🐚 ShellExec                                            │
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ tools/file_tools_bridge.py → executor/shell.py                          │ │
│ │ Step 3: 执行文件读取                                                    │ │
│ │         - 执行命令: cat / head / tail                                   │ │
│ │         - 处理错误: 文件不存在、无权限、二进制文件                        │ │
│ │         - 大文件截断                                                    │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 🧠 Decision Layer / 🤖 LLM                                                  │
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ agent/llm_tool_selector.py                                              │ │
│ │ Step 4: LLM 格式化回复（可选）                                           │ │
│ │         - 将文件内容格式化为易读格式                                      │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 🚪 Access Layer / 🚪 Gateway                                                 │
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ agent/modern_agent.py                                                   │ │
│ │ Step 5: 展示文件内容给用户                                               │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
```

### 支持的命令变体

| 用户输入 | 执行命令 |
|---------|---------|
| "看文件" | `cat` |
| "前N行" | `head -n N` |
| "后N行" | `tail -n N` |
| "行号" | `cat -n` |

### 错误处理

| 场景 | 返回 |
|------|------|
| 文件不存在 | "文件不存在: xxx" |
| 无权限 | "无权限读取: xxx" |
| 二进制文件 | "这是二进制文件，无法显示" |
| 文件过大 | 截断 + "文件过大，仅显示前X行..." |

用户: "帮我创建一个 test.py 文件，内容是 print('hello')"

```
┌──────────────────────────────────────────────────────────────────────────────┐

│ ┌────────────────────────────────────────────────────────────────────────┐ │

│ │ Step 1: LLM 决策                                                        │ │

│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 🏃 Execution Layer / 🛠️ ToolExec                                            │
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ tools/registry.py                                                       │ │
│ │ Step 2: 安全验证                                                        │ │
│ │         - 检查文件路径是否安全                                            │ │

│ │         - 检查文件名合法性                                               │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 🏃 Execution Layer / 🛠️ ToolExec                                            │
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ tools/file_tools_bridge.py                                              │ │
│ │ Step 3: 备份（可选）                                                     │ │
│ │         - mode=overwrite 且文件存在时创建备份                             │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 🏃 Execution Layer / 🐚 ShellExec                                            │
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ executor/shell.py                                                       │ │
│ │ Step 4: 执行写入命令                                                    │ │
│ │         - create: echo "content" > file                                 │ │
│ │         - append: echo "content" >> file                                │ │
│ │         - overwrite: echo "content" > file                            │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌──────────────────────────────────────────────────────────────────────────────┐

│ └────────────────────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────────────┘
```

### 写入模式

| 模式 | 行为 | 用户触发 |
|------|------|---------|
| `create` | 新建文件 | "创建" / "保存为" |
| `append` | 追加内容 | "在末尾添加" / "追加" |
| `overwrite` | 覆盖内容 | "替换内容" / "重写" |
